This project is part of Python Assignment Part 4.

In this assignment, I worked on analysing a student performance dataset and building a simple machine learning model.

The dataset was loaded using pandas and explored by checking basic information, summary statistics, and pass/fail counts. I also calculated average scores and identified the top-performing student.

Different visualizations were created using Matplotlib and Seaborn, such as bar charts, histograms, scatter plots, and box plots to better understand the data.

Finally, a Logistic Regression model was trained to predict whether a student will pass or fail based on their scores, attendance, and study hours. The model was evaluated on test data and used to make predictions for a new student.

This assignment helped me understand the complete workflow from data analysis and visualization to building and evaluating a machine learning model.
